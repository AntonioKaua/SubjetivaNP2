Tarefa 8: Questões sobre Variáveis
Responda as seguintes questões:
Para que servem as variáveis?

    Para armazenar pequenas quantidades de dados em um código, elas não são fixas, por isso se chamam variáveis

Quais tipos de variáveis vimos?

str, char, int, float e bool

Como declarar uma variável?

declare o tipo dela e depois o nome
Ex: int numero;

Se quiser, adicione um valor
Ex: int numero = 4;

O que faz o operador = ?

    Ele atribui um valor a uma variável;
